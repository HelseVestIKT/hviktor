Hviktor Angular MCP

Eksperimentell

Hviktor Angular MCP er en MCP-server som gir AI-assistenter i VS Code tilgang til dokumentasjon og kodeeksempler for Hviktor Angular-komponentene. Serveren kjører i Azure og er tilgjengelig over HTTPS.

Krav

  • VS Code med GitHub Copilot.

1) Konfigurer MCP i VS Code

  1. Åpne Command Palette.

  2. Kjør MCP: Open User Configuration.

  3. Legg inn konfigurasjonen under i mcp.json.

{
  "servers": {
    "hviktorAngularMcp": {
      "type": "http",
      "url": "https://hviktor-angular-mcp.azure.ihelse.net/mcp"
    }
  }
}

Bruk mcp.json, ikke settings.json, for MCP-servere.

2) Start og verifiser

  1. Last VS Code på nytt: Developer: Reload Window.

  2. Kjør MCP: List Servers og bekreft at hviktorAngularMcp har status Running.

  3. Test med en av promptene under i GitHub Copilot Chat.
