Opprett et nytt prosjekt med Angular og Prime ng

  1. npm install -g @angular/cli
  2. ng new my-app
  3. npm install primeng
  4. npm install primeicons
  5. Kopier font-filene i assets\static\ mappa og legg inn i prosjektet ditt.
  6. Kopier style-filene i styles mappa og legg inn i prosjektet ditt.

    7. NB: prismjs er ikkje nødvendig å ta med i prosjektet.
