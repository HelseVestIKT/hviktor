Bibliotek for Hviktor

1. Opprett et nytt prosjekt med Angular

💎 npm install -g @angular/cli

💎 ng new my-app

💎 Would you like to add Angular routing? Yes

💎 Which stylesheet format would you like to use? SCSS

2. Legg til primeng i prosjektet

💎 Gå inn i prosjektet du oppretta: cd my-app

💎 npm install primeng

💎 npm install primeicons

3. Legg inn font fra HVIKTOR

💎 Kopier font-filene i assets\static\ mappa og legg inn i prosjektet ditt.

💎 Trykk på de tre prikkene til høyre for static og last ned som zip.

💎 Pakk ut mappen i assetsmappen til prosjektet ditt.

4. Legg inn styles fra HVIKTOR

💎 Kopier style-filene i styles mappa og legg inn i prosjektet ditt.

💎 Trykk på de tre prikkene til høyre for styles og last ned som zip.

💎 Opprett mappen "styles" under mappen "src".

💎 Pakk den ut i styles-mappen du nettopp opprettet.

5. Oppdater angular.json

💎 Oppdater angular.json med:

"styles": [
        "node_modules/primeicons/primeicons.css",
        "src/styles/prime-theme.scss",
        "node_modules/primeng/resources/primeng.min.css",
        "src/styles/css-reset.scss",
        "src/styles/base.scss",
        "src/styles/styles.scss"
    ],

6. Legg til primeng-komponenter i app.module

💎 Primeng komponentene som du skal bruke i prosjektet, må legges til og importeres i app.module.ts-filen
